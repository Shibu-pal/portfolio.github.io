Project Overview

The Online Examination System is a web-based portal designed to conduct automated multiple-choice tests. Developed as a Major Project for my CST 6th Semester, this application aims to replace traditional paper-based exams with a streamlined, digital alternative.

The Challenge

The primary challenge was building a system that could handle timed examinations while providing instant, accurate feedback to students. We needed a robust way to manage a large pool of questions and ensure that administrative tasks, like enabling or disabling student accounts, could be performed easily.

The Solution

I focused on the core backend architecture using PHP and MySQL. Key implementations included:

  • Session Management: Implemented secure login and session tracking to prevent unauthorized access to exam materials.
  • Automated Grading: Developed a logic that compares student answers against the database in real-time, delivering instant scores upon completion.
  • Administrative Control: Created a comprehensive Admin Panel for question management (Add/Remove) and user account status control.
  • Answer Review System: Built a post-test feature allowing students to review their responses against correct answers for better learning outcomes.
