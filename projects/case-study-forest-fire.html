Project Overview

The Forest Fire Detection System is a sophisticated application designed to identify and predict forest fire risks. It leverages a modern full-stack architecture with Laravel and Inertia.js (React) on the front end, integrated with Python-based Machine Learning models on the back end.

The Challenge

The core challenge was to create a real-time system capable of processing both environmental data (like temperature and humidity) and visual imagery to predict fire outbreaks accurately. Bridging the gap between a PHP-based web framework and Python-based predictive scripts was critical for a seamless user experience.

The Solution

I built a hybrid system that utilizes specialized controllers to communicate between technologies:

  • Dual-Model Prediction: Developed two distinct Python scripts: predict_fire.py for processing environmental data and image_predict_fire.py for image-based detection.
  • Laravel-Python Bridge: Implemented a DataController in Laravel that executes Python processes using Shell commands and parses the JSON output to display results in the React UI.
  • Automated Forecasting: Created a custom Artisan command (RunFirePrediction) to automate prediction cycles.
  • Reactive UI: Used Inertia.js with React and TypeScript to build a fast, single-page application experience for monitoring data.

Technologies Used

Laravel React (TypeScript) Inertia.js Python (Scikit-learn)
