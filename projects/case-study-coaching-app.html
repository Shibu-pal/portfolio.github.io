Project Overview

This project is a comprehensive coaching management platform designed to streamline admission workflows and internal administrative tasks for educational institutes. Built for a specific client, the application centralizes operations by providing distinct modules for administrators, teachers, and students.

The Challenge

The client required an automated solution to replace manual admission processes and speed up the distribution of official documents. The primary technical hurdle was managing multiple user roles (Multi-Auth) while ensuring secure and automated communication channels.

The Solution

Using the Laravel framework, I implemented several key features to meet these requirements:

  • Multi-Role Authentication: Created secure and separate portals for Admins, Teachers, and Students to manage their specific data.
  • Automated Admit Cards: Developed a system to automatically generate admit cards and deliver them to students as PDF attachments via email.
  • Notice Board Management: Built an administrative interface for creating and managing public announcements in real-time.
  • Directory Management: Implemented dynamic lists for faculty and student profiles with role-based data access.

Technologies

Laravel MySQL SASS SMTP/Email API
