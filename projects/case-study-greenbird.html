Green Bird: Peer-to-Peer Exchange

Role: Full Stack Developer Year: 2025 Category: Web Application

[ Insert High-Quality Project Screenshot Here ]

Project Overview

Green Bird is a sustainable marketplace platform built with Laravel designed to facilitate the exchange of pre-loved items between users using a virtual points system. The goal was to reduce waste by encouraging reuse within a local community.

The Challenge

Building a trustworthy marketplace requires secure user identification and a reliable transaction system. I needed to ensure that users were legitimate and that the exchange of "points" happened only when both parties were satisfied.

The Solution

I implemented a multi-layered security and transaction flow:

  • OTP Verification: Integrated a custom 6-digit email OTP system to verify users during registration.
  • Point-Based Transactions: Developed a backend logic that deducts points from the buyer and adds them to the seller only upon confirmation of receipt.
  • Real-time Notifications: Created a notification system to handle meeting requests and pickup times between users.
  • Social Authentication: Integrated Laravel Socialite for seamless login via Google and GitHub.

Technologies Used
