## Features

- **Multi-Page Architecture:** Dedicated pages for Home, About, Projects, and Contact.
- **Dark/Light Mode Toggle:** A smooth sliding switch that remembers your preference using `LocalStorage`.
- **AOS Animations:** Engaging scroll-based animations for a premium feel.
- **Workable Contact Form:** Integrated with Web3Forms (No PHP required for the front-end).
- **Responsive Design:** Fully optimized for Mobile, Tablet, and Desktop views.
- **Interactive Progress Bars:** Showcasing technical proficiency in Laravel, React, and more.
- **Project Case Studies:** Detailed breakdown of core projects like "Crystal Career Institute" and "Forest Fire Detection".

---

## Tech Stack

- **Frontend:** HTML5, CSS3 (Custom Variables, Grid, Flexbox), Vanilla JavaScript
- **Animations:** AOS (Animate On Scroll) Library
- **Icons:** SVG Icons & Font Awesome
- **Forms:** Web3Forms API
- **Fonts:** Google Fonts (Poppins)

## Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Shibu-pal/portfolio.github.io
   ```

2. **Personalize the details:**

- Open the HTML files and update your name, bio, CV Link, and project links.

- Update your Web3Forms Access Key in contact.html.

- Replace the profile picture in the /assets folder.

3. **Run the site:**

- Simply open index.html in any modern web browser or use the Live Server extension in VS Code.
